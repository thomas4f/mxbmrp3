# MXBMRP3 v__VERSION__

A plugin for MX Bikes, GP Bikes, and Kart Racing Pro with customizable on-screen
displays, immersion features, social tools, and streaming/broadcasting overlays.

If you'd rather use the guided installer, download mxbmrp3-Setup.exe
from the GitHub releases page instead.

## Manual Installation

Copy the appropriate DLO file and mxbmrp3_data\ folder to [Game]\plugins\

- MX Bikes:        mxbmrp3.dlo
- GP Bikes:        mxbmrp3_gpb.dlo
- Kart Racing Pro: mxbmrp3_krp.dlo

    [Game]\
    +---plugins\
        +-- mxbmrp3_data\        <-- Add this folder
        +-- mxbmrp3.dlo          <-- Add this (MX Bikes only)
        +-- mxbmrp3_gpb.dlo      <-- Add this (GP Bikes only)
        +-- mxbmrp3_krp.dlo      <-- Add this (Kart Racing Pro only)

## Notes

- Do NOT delete native game files (proxy64.dlo, proxy_udp64.dlo, xinput64.dli,
  or telemetry64.dlo for GP Bikes)
- Each game needs its own copy of the mxbmrp3_data folder
- Settings are stored separately per game in Documents\PiBoSo\[Game]\mxbmrp3\
