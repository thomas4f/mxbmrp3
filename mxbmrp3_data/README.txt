# MXBMRP3 - Plugin Data Folder

Do NOT edit files in this folder.

This folder is part of the plugin distribution and its contents will be
overwritten when you update MXBMRP3. Any changes you make here (custom
fonts, icons, textures, web overlay tweaks, etc.) will be lost.

## Customization

To customize the plugin without losing your changes on update, place
your files in the user override folder instead:

    Documents\PiBoSo\[Game]\mxbmrp3\

Files there are loaded in addition to (and take precedence over) the
ones shipped here, and they are preserved across plugin updates.

For details on what can be customized and how, visit:
https://thomas4f.github.io/mxbmrp3/#modding
